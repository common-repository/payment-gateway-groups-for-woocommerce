Allows you to create groups for payment gateways on the checkout page.

== Description ==
Payment Gateway Groups for WooCommerce allows you to create groups from your list of available payment gateways. Tested with popular themes. There is a normal mod and toggle. You can also change the active color. 

Useful if you have a large number of payment gateways.

[Pro Version](https://businessupwebsite.com/plugin/payment-gateway-groups-for-woocommerce/)
*	Drag and Drop
*	Infinite Groups
*	Subgroups
*	Image titles

**!Warning** Plugin uses WooCommerce page templates ("checkout/payment.php" and "checkout/payment-method.php"). If there are similar files in your "theme-name/woocommerce" folder, then they will not work. 

== Installation ==

= Minimum Requirements =

* WooCommerce 3.5.3 or greater

1. Upload the entire plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to "WooCoommerce" -> "Settings" -> "Payment Gateway Groups".
